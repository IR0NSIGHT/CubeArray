CubeArray - Minecraft Schematic Viewer
========================================

WHAT IS THIS?
CubeArray lets you view and explore 3D models of Minecraft schematics.

HOW TO RUN
- Double-click CubeArray.exe      (easiest - no setup needed)
- OR double-click CubeArray.jar    (only if you have Java 17 installed)

If CubeArray.exe does nothing or shows a security warning, click "More info" then "Run anyway".
